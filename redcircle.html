10章

10.1

餌となりうるものが n 種あり、 i = 1, 2,\cdots, n といった様にそれぞれ区別される

これら i 種の餌に 出会う確率を \lambda_i とする。 i 種の餌を食べると カロリー g_i が得られるが、食べ始めてから次の探索を行うまでに 処理時間 h_i が必要になる。

餌によってはコスパが悪い（ g_i に対して h_i が大きい）ため、探索者は食べる時もあれば食べない時もある。 そのため i 種の餌それぞれに対し 出会った時に食べる確率の組 (p_1, p_2, \cdots, p_n) を定義する。

探索し出会って (\lambda_i)、食べるか判断し (p_i)、処理する (h_i) というサイクルを繰り返す場合、捕食速度は（つまり \lambda_i あたりのカロリー g_ip_i は）1サイクルあたりの平均カロリーを1サイクルあたりの平均所要時間で割れば求まる。

まず1サイクルで出会う餌が i 種である確率は \lambda_i / \sum\limits_i \lambda_i であり、そのカロリーは g_i である。i 種の期待カロリーは、確率をかけて表すことができ \frac {\lambda_i}{\sum\limits_i \lambda_i}p_ig_i 出会う i 種すべての総和をとって \frac {\sum\limits_i\lambda_i p_i \boldsymbol g_i} {\sum\limits_i \lambda_i} 1サイクル中の期待カロリーとすることができる。

次に1サイクル中で得た餌 n 種の期待処理時間もカロリーと同じ様に計算できる。 \frac {\sum\limits_i\lambda_i p_i \boldsymbol h_i} {\sum\limits_i \lambda_i}

ただし所要時間として1サイクルの開始（つまり探索の開始)から、餌を見つけるまでの時も考慮しなければならない。 開始してから n 種のどれかに会うまでの確率は単位時間あたり 1/\sum\limits_i で表すことができる。

これらより、期待カロリーを期待時間で割ると求まる捕食速度 r は以下の様に書ける。

r(p_1, p_2, \cdots, p_n) = \frac { \dfrac{\sum\limits_i \lambda_i p_i g_i } {\sum\limits_i\lambda_i}} { \dfrac{1}{\sum\limits_i \lambda_i} + \dfrac{\sum\limits_i\lambda_i p_i h_i} {\sum\limits_i \lambda_i}} \\

r を最大にする食べ方(各餌の p の最適な割り振り)を求めるために p_k を微分すると \partial r/\partial p_k = \bigg\{ \dfrac {\lambda_k h_k} {1 + \sum\limits_{i} \lambda_i p_i h_i} \bigg\} \bigg\{ \dfrac {g_k}{h_k}-r \bigg\} この式から \dfrac {g_k}{h_k} が r より大きい餌は食べられ、r より小さいと食べられない。

会う頻度 \lambda_i が関係ないのが面白い。

存在する n 種の餌を \dfrac {g_i}{h_i} の値が大きい順に並べていくと \dfrac {g_1}{h_1} > \dfrac {g_2}{h_2} > \cdots > \dfrac {g_k}{h_k} > r > \dfrac {g_{k+1}}{h_{k+1}} > \dots となり r より小さい期待カロリーが小さい餌は食べない、すなわち \boldsymbol p_i \boldsymbol = \boldsymbol 0 となる。

実際に動物を用いた実験では、大きい餌に専念する利益が小さい場合には専食する割合が非常に少ない。




10.2生物が適応的行動を取る理由

餌探しのコストを小さくできれば繁殖行動などにリソースを費やすことができ、繁殖成功度（適応度）を上げることができる。

動物行動の遺伝的基盤は環境中のデータを元に餌選択を行うための方法を規定していて、それは適応的な方向に進化している。

前節の捕食速度 r は評価関数と呼ばれ最適化の基礎的な尺度の一つであり合理的意思決定に役立つ。


10.3 採餌場所の決定

10.1の餌選択モデルでは、さまざまな餌が完全に混合していて（確率に基づき）ランダムに出会うと仮定した。

変わって、距離が離れた別々の2点にそれぞれ別の昆虫1と2がいる。この場合、同時に餌を狙うことができないため片方の餌の捕食速度がもう片方より大きい場合にのみ採餌が続く。

つまり \dfrac {\lambda_1 g_1}{1 + \lambda_1 h_1} > \dfrac {\lambda_2 g_2}{1 + \lambda_2 h_2} の場合にのみ1種の昆虫が好まれる。

そのため、ある餌の密度が高くなると急にそれが食べられ始める。

10.4 餌の探し方

10.3の様に、鳥は餌の昆虫がいる木（パッチ）を飛び回っている。

それぞれのパッチでは餌がなくなるまで探すべきか、適当に切り上げるべきかを考える。

昆虫の数や見つかりやすさの異なる n 種のパッチを考える。1つのパッチで得られる餌の量は、探索時間 ( t_i ) の増加関数 ( G_i(t_i) ) で与えられる。また1つのパッチ内での捕食速度は時間と共に減少する。

これらから、10.1の各式を利用して、餌をパッチに置き換えることで表せる。

まず p_i (食べるかどうかの確率)は用いず、 h_i はパッチ内での探索時間に含まれるためこれらは除く。

なので期待捕獲量と期待探索時間が \begin{gather} \frac {\sum\limits_i\lambda_i \boldsymbol {G_i(t_i) }} {\sum\limits_i \lambda_i} \\ \dfrac {1}{\sum\limits_i\lambda_i} + \frac {\sum\limits_i \lambda_i \boldsymbol {t_i}} {\sum\limits_i \lambda_i} \end{gather} とそれぞれ表せ

r(t_1, t_2, \cdots, t_n) = \frac { \dfrac{\sum\limits_i \lambda_i \boldsymbol {G_i(t_i)} } {\sum\limits_i\lambda_i}} { \dfrac{1}{\sum\limits_i \lambda_i} + \dfrac{\sum\limits_i\lambda_i \boldsymbol {t_i}} {\sum\limits_i \lambda_i}} \\ となる。

この捕食速度 r を t_i \ge 0, \dots, t_n \ge 0 という制約のもとで最大にするものを探すために t_i について微分する。

r を最大にする解が、t_1 \ge 0, \dots, t_k \ge 0, t_{k+1} = \dots = t_n = 0 だったとすると

\begin{gather} \dfrac {dG_i(t_i)}{dt} = r & i = 1, 2,\dots, k \\ \dfrac {dG_i(0)}{dt} < r & i = k+1,\dots, n \end{gather} が成立。

捕食速度が減速していき、平均 r を下回る場合は現在のパッチを放棄して新たなパッチに移る方が良い。

パッチ間の距離が長くなり 1 / \lambda_i が大きくなれば期待捕食速度 r の値が下がるため、パッチで探索する時間が長くなる。

10.5 収量変動の効果

多くの状況で生物は 収量の平均が同じでもばらつきが少ない場所を好む。これはリスク回避的行動と言われる。例えば摂食量の増分に対して適応度がの増分が緩やかになっていく場合、餌の量が増えても適応度があまり変化しない。

この時に適応度の平均裏を尺度とすると、餌の量がばらつかない方が適応度の平均値的に安定であるため餌場として適当であると考えることができる。

11章 成長と繁殖の戦略

11.1 卵数と卵サイズ

卵のサイズが小さいと体サイズは小さく、生活史の初期段階も長いため死亡率が高くなるが繁殖に割ける資源は限られているため大きい卵は数を産むことができない。

1卵の重量を x、卵数を y とする。

子の生存率を卵サイズ x の増加関数として S(x) とすると、子の期待生存率は \phi = y S(x) と表せる。さらに卵を生産するのに親が使える資源を xy = \rho と考える。

この制約のもと \phi を最大にする x と y を求める。

制約条件 xy = \rho を代入して、 \phi = \dfrac {\rho S(x)}{x} 。

卵サイズと生存率の関係を表す曲線は、卵サイズが大きくなるほど生存率が上がることが考えられる（S字とか）。 \dfrac { S(x)}{x} は卵サイズと生存率をグラフにしたときに原点から伸びる線分の傾きであるため、この線が卵サイズ-生存率曲線に接する点が最適卵サイズ x^* になる。

なので \phi を x で微分すると \dfrac {d \bigg\{ \dfrac {S(x)}{x} \bigg\}} {dx} = 0 より

\dfrac {x^*dS(x)}{dx} = S(x^*) が導かれて、最適卵サイズがもとまる。

ラグランジュ

未知の定数 \lambda と制約条件を用いて \Phi = yS(x) + \lambda (\rho-xy) これを x と y それぞれについて偏微分してそれを0とおいた \begin{gather} \dfrac {\partial \Phi}{\partial x} = \dfrac {y\ dS(x)}{dx} - \lambda y = 0 \\ \dfrac {\partial \Phi}{\partial y} = S(x) - \lambda x = 0 \end{gather} これらの式から \lambda を消すと \dfrac {x^*dS(x)}{dx} = S(x^*) が導かれる。

最適卵サイズ x^*　は、\rho (つまり資源の総量)には依存せず生存率 S(x) によって決まるので大きな卵を産む方が有利になる。


11.2 陸上植物の地上部/地下部のバランス

植物の器官のサイズは環境に対してバランスを保って変化する。得られる水分量が多いと枝葉は大きく成長するが、地下の根は貧弱になる。逆に栄養塩類や水分が不足して日光などが十分にある場合、地上部が貧弱になるが根などは成長する。

光合成による物質生産において光吸収は葉によって行われるため、光合成速度は地上部の成長に比例する。また根も光合成に必要な水や栄養を吸収するため、根の成長も光合成に寄与する

一日あたりの純光合成の速度を、地上部サイズ x と地下部サイズ y を変数とする増加関数 g(x, y) で表す。

さらに、L を地上部の光 W を土壌の含水量とする。

これらから以下の式を定義する。 g(x, y) = \dfrac {1}{\dfrac{a}{Lx^b} + \dfrac{c}{Wy^d}} 植物の総重量が一定の x+y = \rho という条件下で純光合成速度が最大になるものをラグランジュの未定乗数によって解くと [地下部] \propto [光 / 土壌成分]^ \frac {1}{(d+1)} [地上部]^ \frac {(b+1)}{(d+1)} となる。

異なる器官のサイズの対数が線形関係を満たす成長スキームということを示している。


11.3 1回繁殖か繰り返し繁殖か

無性生殖する生物について考える。

長い時間が経つと、安定齢分布に達して密度が時間 t に依存して指数増加していく。

この時の人口増加率 r がマルサス係数で、離散時間の齢分布モデルの \lambda を置き換えた場合

1 = \sum_{x=0} e^{-r(x+1)}\ l_x\ m_x となる。l_x は子の x 歳までの生存率、m_x は x 歳での出産率。

毎年同じ率で繁殖し続ける生物を考える。

雌1匹から生まれる子数が N でその半分が雌で翌年までの生存率が q なら m_0 = m_1 = \dots = \frac {qN}{2} 。成体の年間生存率も等しく p_0 = p_1 = \dots = p の場合、マルサス係数 r は

1 = \sum_{x=0} e^{-r(x+1)} p^i \frac {qN}{2} = \frac {qN}{2e^r(\frac {1-p}{e^r})} を満たすもので r = log(\frac {qN}{2+p}) になる。

最初の繁殖時に全ての成体が死ぬタイプ (p = 0) と、生き残り繁殖を続けるタイプ (p > 0)がいて前者のタイプの方が子供の数が多いとする。マルサス係数を比較すると、子の数 N の差が \frac {2p}{q}を超えると一回繁殖が有利になる。

11.4 なぜマルサス係数が進化の基準として使えるのか

1つの懸念点として雌個体から生まれた子が全て親と同一の生活史を持つと考えているが、ある遺伝子型の雌が親が荒廃したのと同じ遺伝子タイプを持つ個体と必ず交配するという仮定は置けない。しかし実際には、雌雄の齢別周産率が比例していて任意交配が仮定できる場合は、先述のモデルが適用できることがわかっている。

つまり異なる遺伝子型タイプの l_x と m_x からマルサス係数を比較することで集団内で遺伝子型が増えるかどうかという、適応度に関する議論ができる。

もう1つは生存率・出産率が個体の密度に影響されないのかという点である。この仮定では個体数は増大するが、実際には定常状態にある場合(あるタイプが集団内で優占している場合)マルサス係数は0になる。

優占しているタイプが進化的に安定で変異の拡大を許さない場合は変異を持ったタイプが入り込んできたとしてもそのタイプは減少し消滅する、つまりマルサス係数が負と捉えることができる。そのため平衡状態では優占種のマルサス係数が他のどのタイプよりも大きいということができる。


11.5老衰の進化

出産率を上げると生存率を維持するためには相応の資源が必要になる。

例として生存率に対する感度の式を用いる。 \dfrac {\partial \lambda}{\partial m_k} =\dfrac {1}{p_k} \sum_{a \ge k+1} \dfrac {m_a l_a}{\lambda^a T_g}

この式では生存率は繁殖開始齢以前に大きく繁殖齢後はほとんど0になる。

そのため若齢の生存率をあげる方が適応度の改善が期待でき、ある死亡要因発現を高齢期にずらせるような遺伝子が集団中に広がる。

また別の例として k 歳での生存率と出産率に対する r の感度の比は \dfrac{\dfrac {\partial \lambda}{\partial p_k}} {\dfrac {\partial \lambda}{\partial m_k}} = \dfrac{v_{k+1}}{v_0} となって k+1 歳での繁殖価（産むことができる子の総数を生まれた時期で調整して算出した値）に比例している。

老齢になると繁殖価が減少するので、現在での繁殖が非常に重要。


11.6 生涯の繁殖成功度

マルサス係数は11.10式から 1 = \sum_{x=0} e^{-r(x+1)}\ l_x\ m_x

と示されたが、陰関数なので扱いにくい。そのため生まれた1個体が生涯に生産する次世代の平均数である純繁殖率を用いる。m_x は出産率 l_x は 生存率である。 R_0 = \sum^{\infty}_{x=0} p_0\dots p_{x-1}\ m_x = \sum^{\infty}_{x=0} l_x m_x さらに年齢ごとに行われる繁殖効果の違いを補正するため、集団サイズが急激に増大している時には遅い繁殖の価値を年あたり増加率 e^r によって割り引く。

f(s, r) = \sum^{\infty}_{x=0} p_0(s)\dots p_{x-1(s)}\ m_x(s)\ e^{-r(x+1)} これらから齢ごとの生存率 px(s) や出産率 m_x(s) が戦略 s に左右されることを示した。

数学的にはマルサス係数 r を最大にすることと生涯の繁殖成功度 f(s,r) を最大にすることは等価で

戦略s* はマルサス係数 r(s) を最大にする。この時 r^* = r(s*) とおくと、$f(s, r) $ は s の関数として s^* の時最大になる。
逆に s が f(s, r) を最大にするものならそれはマルサス係数 r を最大にするものである。

密度などに影響を受ける平衡状態では r^* = 0 が成り立ち生涯の繁殖成功度 f(s,0) になる。
